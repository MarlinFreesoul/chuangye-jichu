## 一、执行摘要（Executive Summary）

> 本章节是整份计划书的浓缩精华，用于快速传达项目核心价值。建议 300–500 字，覆盖以下全部要点。

- [ ] **项目名称与 LOGO 概念**：简述项目名称及品牌定位
- [ ] **创业团队**：创始人及核心成员简介（3–5 人）
- [ ] **痛点问题**：目标用户面临的具体问题
- [ ] **解决方案**：产品/服务的核心价值主张
- [ ] **市场规模**：目标市场的容量与增长趋势
- [ ] **商业模式**：如何盈利（收入来源、定价策略）
- [ ] **竞争优势**：与竞争对手的差异化壁垒
- [ ] **融资需求**：本轮融资额度、估值、资金用途
- [ ] **发展愿景**：3–5 年战略目标

## 四、创业机会识别与评估

> 本章对应课程第三章——从海量信息中识别商业机会，科学评估机会价值。

### 4.1 创业机会的来源

- [ ] 政策驱动型机会（政府补贴、行业规划）
- [ ] 技术变革型机会（AI、区块链、新能源）
- [ ] 市场缺口型机会（细分人群、未被满足的需求）
- [ ] 社会变迁型机会（老龄化、环保意识崛起）

### 4.2 创业机会的识别方法

- [ ] 痛点分析法（用户旅程图）
- [ ] 趋势分析法（PEST 分析框架）
- [ ] 跨界融合法（行业交叉创新）
- [ ] 痛点 → 解决方案 → 市场验证的思维链

### 4.3 创业机会的评估模型

- [ ] Timmons 模型：机会、市场、团队三要素评估
- [ ] AFSAS 模型：吸引力、可行性、可扩展性评估
- [ ] 市场规模评估（SAM / TAM / SOM）
- [ ] 机会进入时机分析（是太早、太晚，还是刚刚好？）

## 五、创业项目选择与产品/服务设计

> 本章对应课程第四章——将创意转化为可落地的商业项目。

### 5.1 项目选择原则

- [ ] 政策契合度（符合国家产业方向）

- [ ] 用户旅程地图：触达 → 认知 → 使用 → 复购

### 5.3 核心技术或运营壁垒

- [ ] 技术专利或算法壁垒
- [ ] 独家渠道或供应链优势
- [ ] 品牌认知与用户黏性壁垒
- [ ] 网络效应或规模效应壁垒

## 七、市场分析与竞争策略

> 本章对应课程第六章——深度理解市场，制定竞争战略。

### 7.1 宏观市场分析

- [ ] PEST 分析：政治/经济/社会/技术四维度
- [ ] 行业生命周期阶段（萌芽期 / 成长期 / 成熟期 / 衰退期）
- [ ] 政策环境对行业的影响

### 7.2 目标市场规模

- [ ] **TAM（潜在市场总量）**：最大可触达市场
- [ ] **SAM（可服务市场）**：目标区域/细分市场
- [ ] **SOM（可获得市场）**：3 年内实际可获得的市场份额
- [ ] 市场增长率预测（CAGR）

### 7.3 竞争格局分析

- [ ] 波特五力模型分析（供应商议价能力 / 买方议价能力 / 新进入者威胁 / 替代品威胁 / 行业内竞争）
- [ ] 主要竞争对手分析（至少 3 家，SWOT 对比表）
- [ ] 竞争战略选择：差异化 / 成本领先 / 聚焦细分市场

### 7.4 市场营销策略（4P + 4C）

#### 产品（Product）
- [ ] 产品线规划（引流款 / 利润款 / 形象款）
- [ ] 产品迭代路线图（V1.0 → V2.0 → V3.0）

#### 价格（Price）
- [ ] 价格体系设计
- [ ] 促销活动规划（节日营销、新用户优惠）

#### 渠道（Place）
- [ ] 线上渠道：微信生态 / 电商平台 / 自有 App
- [ ] 线下渠道：直营店 / 加盟店 / 代理商
- [ ] O2O 全渠道融合策略

#### 推广（Promotion）
- [ ] 内容营销策略（短视频 / 直播 / 图文种草）
- [ ] KOL / KOC 合作推广
- [ ] 私域流量运营（微信社群、企业微信）
- [ ] 用户裂变机制设计（邀请有礼、拼团、分销）

## 八、创业融资与财务规划

> 本章对应课程第七章——理解创业融资渠道，做好财务预测。

### 8.1 创业融资渠道

- [ ] **股权融资**：天使投资 / VC（风险投资）/ 股权众筹
- [ ] **债权融资**：银行贷款 / 政策贴息贷款 / 创业担保贷款
- [ ] **政府补贴**：大学生创业扶持资金、科技创新基金

### 8.2 融资规划

- [ ] 本轮融资目标金额

- [ ] 出让股权比例
- [ ] 资金用途分配明细（研发 / 市场 / 运营 / 人员）
- [ ] 下轮融资里程碑规划

### 8.3 财务预测（3 年）

#### 8.3.1 启动成本估算

#### 8.3.3 成本结构分析

## 十、创业风险管理

> 本章对应课程第九章——识别风险，建立预警与应对机制。

### 10.1 风险识别

- [ ] **市场风险**：需求不足、竞争加剧、市场变化
- [ ] **财务风险**：资金链断裂、回款周期长、成本超支

- [ ] **政策风险**：行业监管变化、政策取消

| 运营风险 |  |  |  |  |
| 政策风险 |  |  |  |  |

### 10.3 风险应对策略
